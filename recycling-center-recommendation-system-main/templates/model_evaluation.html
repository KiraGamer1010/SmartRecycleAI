Performance analysis

Academic interpretation

Accuracy describes the global share of correct predictions, while precision and recall show how reliable the model is when assigning each priority class. F1-score balances precision and recall, making it a strong selection metric for an imbalanced municipal planning target.

Confusion matrices should be reviewed before field use because a false low-priority assignment can delay infrastructure attention. The selected model should support expert review, not replace it.

Reusable evidence

Stored evaluation files

Metrics are saved in `reports/model_evaluation/model_metrics.csv`, full evaluation context is saved as JSON, and all charts are stored under `static/generated/model_evaluation`.

This structure keeps the evaluation reusable for screenshots, PDF reporting, Render deployment, and future model comparison.
