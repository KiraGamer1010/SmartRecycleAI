Environmental problem

Why the project matters

Recycling infrastructure planning is a Machine Learning problem because municipalities are not equal: they differ in population pressure, recyclable waste generation, collection coverage, facility capacity, accessibility, social vulnerability, and hazardous waste exposure. SmartRecycleAI turns those conditions into measurable indicators so decisions can be compared with consistent analytical context.

Demand Pressure

Municipalities with more recyclable material need stronger collection and processing capacity.

Access Gap

Distance, coverage, and existing center availability affect whether citizens can participate in recovery systems.

Planning Intelligence

The platform connects raw datasets, engineered features, models, charts, and prediction results in one academic portal.

CRISP-ML phase map

How the application is organized

1

Business Understanding

Defines the recycling planning problem, objectives, stakeholders, risks, and expected impact.

2

Data Understanding

Explains source datasets, variable types, exploratory analysis, missing values, and statistical behavior.

3

Data Engineering

Documents cleaning, integration, feature engineering, target encoding, scaling, and processed artifacts.

4

Model Engineering

Trains Logistic Regression, Decision Tree, and Random Forest using a reproducible supervised pipeline.

5

Model Evaluation

Compares accuracy, precision, recall, F1-score, confusion matrices, and ROC curves on test data.

6

Prediction System

Loads the selected model and produces class predictions, probabilities, and confidence scores.

Final system capabilities

Supervised, unsupervised, and operational outputs

Capability Implementation Teacher-friendly interpretation
Supervised classification Logistic Regression, Decision Tree, and Random Forest. Predicts municipal recycling priority classes using engineered indicators.
Unsupervised clustering K-Means over standardized planning features. Discovers groups of municipalities with similar infrastructure and sustainability profiles.
Prediction workflow Flask form, validation, model artifact loading, probabilities, and confidence score. Demonstrates how academic ML code becomes a usable decision-support system.
