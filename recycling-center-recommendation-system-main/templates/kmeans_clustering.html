Unsupervised learning explanation

What K-Means contributes to the academic project

K-Means is used as supporting unsupervised analysis. While supervised models predict known priority classes, clustering discovers similarity profiles without using the target label. This helps the teacher see whether municipalities naturally group according to demand, access, infrastructure, vulnerability, and environmental pressure.

Planning Profiles

Clusters summarize groups of municipalities that require similar recycling infrastructure strategies.

Scaling Requirement

Standardization is necessary because population, waste, distance, poverty, and capacity use different units.

Interpretation Limit

Clusters are exploratory patterns; they do not prove causality or replace supervised evaluation.

Model justification

Why K-Means fits SmartRecycleAI

SmartRecycleAI uses unsupervised learning because the platform does not have a verified historical ground-truth label for every possible recycling center investment decision. K-Means is appropriate here because it groups municipalities by similarity across standardized planning indicators, producing interpretable profiles for sustainability analysis and public policy discussion.

No Complete Labels

Municipal investment priority is a planning decision, not a fully observed historical target.

Similarity Discovery

K-Means groups municipalities with comparable waste, access, vulnerability, and infrastructure patterns.

Transparent Centroids

Cluster centroids can be interpreted as average municipal profiles, which supports academic auditability.

Sustainability Planning

Clusters help compare where environmental pressure and service gaps require coordinated intervention.

WCSS and Inertia Explanation

WCSS means within-cluster sum of squares. In scikit-learn K-Means, this value is reported as inertia. It measures how close municipalities are to their assigned cluster centroid.

WCSS always decreases as K increases, so it cannot be interpreted alone. SmartRecycleAI combines WCSS with Silhouette Score, Davies-Bouldin Index, and academic interpretability.

Elbow Method and K Value Explanation

The elbow method compares within-cluster inertia across candidate K values. The selected value balances compact clusters with a cluster count that remains understandable for municipal planning.

For SmartRecycleAI, K={{ kmeans_analysis.selected_k }} separates municipalities into strategic planning profiles instead of treating all priority zones as one homogeneous group.

Environmental and Operational Implications

K-Means does not replace expert judgment; it groups municipalities with similar pressure profiles so decision makers can compare zones more systematically. High-need clusters indicate where recyclable waste recovery, access improvement, social equity, and environmental risk reduction can be addressed together through new recycling centers or reinforced collection capacity.

The generated artifacts are saved as processed CSV files, allowing the clustering results to be audited, reused, and extended with future supervised recommendation models.

Model limitations

Responsible interpretation boundaries

The clustering depends on feature quality, preprocessing assumptions, and the completeness of the source datasets. Unsupervised learning discovers similarity groups; it does not prove causal impact or replace municipal field validation.

Dataset bias, inconsistent municipal reporting, changing recycling infrastructure, evolving waste generation patterns, and missing geographic geometry can affect interpretation. Results should be reviewed with domain experts before investment decisions.

Future work

Next technical improvements

Future versions can add supervised recommendation models, predictive waste forecasting, geospatial optimization, IoT collection telemetry, real-time monitoring, route optimization, and reinforcement learning for adaptive collection policies.

A production extension should also integrate municipality boundary files, cost constraints, facility travel times, and feedback from environmental or municipal specialists.
