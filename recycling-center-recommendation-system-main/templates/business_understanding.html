Phase 1

Business Understanding

SmartRecycleAI addresses the need for evidence-based recycling infrastructure planning by identifying municipalities where access gaps, waste potential, vulnerability, and infrastructure limitations justify stronger intervention.

Problem Context

Recycling systems often face uneven infrastructure distribution, incomplete local capacity, limited collection coverage, and insufficient prioritization criteria. Municipalities with high recyclable waste potential can remain underserved when planning relies only on administrative availability rather than integrated data.

Business Objective

Build a transparent analytical system that prepares data for recommending where recycling collection centers should be strengthened or created, while supporting academic review of every data decision.

Decision questions

Questions the model should answer

Where is intervention most urgent?

Classify municipalities by recommendation priority using access, waste, and vulnerability features.

Where is infrastructure insufficient?

Compare recycling potential with existing collection centers and registered facility capacity.

Which factors drive priority?

Analyze correlations among poverty, collection coverage, distance, waste generation, and facility availability.

Feasibility

The project is feasible because the repository already contains municipal, census, recycling facility, waste, and projection datasets. The implemented pipeline converts these sources into repeatable processed artifacts, charts, and a model-ready feature matrix without requiring external services.

Flask is appropriate for local academic review and Render deployment because the data volume is small enough for startup-time artifact generation and static chart delivery.

Risks and Limitations

The current target labels come from the existing modeling dataset, so future validation should compare predictions against expert planning decisions. Some sources have different geographic granularity, and the hazardous waste file does not include a department column. The pipeline documents these limitations and prevents ambiguous joins where possible.

Recycling center recommendation should support decision makers, not replace field validation, regulatory review, budget constraints, or community consultation.

Expected value

Impact analysis

Environmental Impact

Improves visibility into recyclable waste potential and hazardous waste pressure, supporting more sustainable material recovery planning.

Operational Impact

Helps compare collection gaps with facility availability, allowing planners to focus on municipalities with measurable infrastructure needs.

Social Impact

Incorporates vulnerability indicators so recommendations account for communities where service access and environmental burden matter most.
