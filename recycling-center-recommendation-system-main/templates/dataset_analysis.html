Dataset overview

What the processed dataset represents

The modeling dataset is the final analytical table produced by the data pipeline. Each row represents a municipality profile, and each column contributes either descriptive context, engineered ML evidence, or the target label used by supervised classifiers.

Priority Distribution

The priority chart shows how target classes are distributed and helps detect imbalance before model evaluation.

Statistical Evidence

Summary charts and tables explain the scale, spread, and relationships of variables used by the models.

Model Relationship

The same processed dataset supports supervised classification, K-Means clustering, and the prediction form defaults.

Evidence What it shows Why it matters
